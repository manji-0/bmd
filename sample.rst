テーブル
--------

reST の grid table は ASCII セルで記述します（日本語セルは Markdown / AsciiDoc を参照）。

+--------------+----------------------+------------+
| Name         | Description          | Version    |
+==============+======================+============+
| ratatui      | TUI framework        | 0.30       |
+--------------+----------------------+------------+
| pulldown-cmark | Markdown parser    | 0.13       |
+--------------+----------------------+------------+
| merman       | Mermaid renderer     | 0.6        |
+--------------+----------------------+------------+

+------+--------------------------------+------+
| Col1 | Long description column        | Val  |
+======+================================+======+
| A    | Wrapping test cell             | 1    |
+------+--------------------------------+------+
| B    | Another row for layout check   | 2    |
+------+--------------------------------+------+
