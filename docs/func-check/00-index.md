## クイック動作確認（5 分）

1. このページから [見出し](./01-headings.md) を `o` で開く
2. `}` で次の見出しへ、`{` で戻る
3. `O` でこの目次に戻る
4. `/` で `機能チェック` を検索し `n` で次マッチへ
5. `h` でヘルプを表示し `Esc` で閉じる

以上で基本ナビゲーションのスモークテストが完了します。
