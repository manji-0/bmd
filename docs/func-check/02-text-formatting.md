# テキスト装飾

[← 目次](./00-index.md)

## インライン装飾

通常のテキストに **太字（Strong）**、*斜体（Emphasis）*、***太字斜体***、および `インラインコード` を混在させた段落です。

`console.log("hello")` のようなコード片は等幅フォントで表示されます。

## 複合装飾

- **太字の中に *斜体* を含む**
- *斜体の中に **太字** を含む*
- リンク風の装飾は [bmd README](../../README.md) を参照（ドキュメントリンク）

## ハードブレイク

次の行は行末にスペース 2 つでハードブレイクです。
この文は同じ段落内の次の行に続きます（ソフトブレイク）。

次の行は空行で区切られた別段落です。

## 長い段落の折り返し

これはターミナル幅に合わせて折り返される長い段落のテストです。英語と日本語が混在しています。The quick brown fox jumps over the lazy dog. 速い茶色の狐が怠けた犬を飛び越える。折り返し位置が単語境界または CJK 文字境界で自然であることを確認してください。

## 確認項目

- [ ] 太字・斜体・インラインコードが正しく着色される
- [ ] ネストした装飾が崩れない
- [ ] ハードブレイク（2 スペース）で改行される
- [ ] 長文が端末幅で折り返される
