# bmd 機能チェック用ドキュメント

手動で bmd の各機能を確認するための Markdown ドキュメント群です。

## 起動方法

```bash
# リポジトリルートから
devbox run run -- docs/func-check/00-index.md

# またはビルド済みバイナリ
./target/release/bmd docs/func-check/00-index.md
```

## 推奨環境

- Kitty / Ghostty / iTerm2 / WezTerm など画像プロトコル対応ターミナル（Mermaid・画像のインライン表示）
- ターミナル幅 80 列以上（テーブル折り返しの確認用）

## チェックリスト

各ドキュメントを開き、表の項目を順に確認してください。`[ ]` は未確認、`[x]` は確認済みとして記入できます（bmd 上のトグルはセッション内のみで、ファイルには保存されません）。

## キーバインド早見表

| キー | 動作 |
|------|------|
| `j` / `k` | 2 行スクロール |
| `d` / `u` | 半ページスクロール |
| `g` / `G` | 先頭 / 末尾 |
| `{` / `}` | 前 / 次の見出し |
| `/` / `?` | 前方 / 後方検索 |
| `n` / `N` | 次 / 前のリンク（検索中はマッチ） |
| `o` / `Enter` | リンクを開く / プレビュー |
| `O` | アンカーまたは前のファイルへ戻る |
| `Esc` | 検索解除 / プレビュー閉じる / スタックリセット |
| `x` | 画面上部のタスク項目をトグル |
| `h` | ヘルプ |
| `q` | 終了 |
