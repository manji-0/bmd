# 画像表示

[← 目次](./00-index.md)

ローカル画像は Mermaid と同様にターミナル画像プロトコルで表示されます。相対パスはこのファイルのディレクトリ基準で解決されます。

## インライン画像

![テスト画像（8x8 赤）](./assets/test-image.png)

## プレビュー

画像リンクはマゼンタ色で表示されます。`n` で画像リンクを選択し `o` でフローティングプレビューを開いてください。

## 存在しない画像

![存在しないファイル](./assets/missing-image.png)

存在しないパスはエラー表示またはプレースホルダになることを確認します（実装に応じて）。

## 確認項目

- [ ] `assets/test-image.png` がインライン表示される
- [ ] 画像リンクがマゼンタで表示される
- [ ] `o` でプレビューオーバーレイが開く
- [ ] `Esc` でプレビューが閉じる
- [ ] 存在しない画像で適切なエラー表示がある
