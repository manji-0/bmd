# リンクとアンカー

[← 目次](./00-index.md)

リンク種別ごとの動作を確認します。`Tab` / `n` でリンクを選択し、`o` または `Enter` で開きます。

## Web リンク（青色）

- [GitHub](https://github.com)
- [Rust 公式](https://www.rust-lang.org)
- [mailto リンク](mailto:test@example.com)（外部ハンドラ）

`o` でブラウザまたはメールクライアントが起動することを確認してください。

## アンカーリンク（同一ドキュメント内）

現在位置: [ジャンプ先セクションへ](#ジャンプ先セクション)

スクロールを上に戻したあと、上のリンクを `o` で開くと下のセクションへジャンプします。

### 別セクション

ここは中間セクションです。`O` でジャンプ前の位置へ戻れます。

### ジャンプ先セクション

アンカー `ジャンプ先セクション` のスラッグは `ジャンプ先セクション`（CJK はそのまま小文字化）です。

別形式: [Foo Bar へ](#foo-bar)

#### Foo Bar

この見出しへのアンカーは `foo-bar` です。

## リンク選択の挙動

画面上に表示されているリンクのみが `n` / `N` の対象です。画面外のリンクはスクロールしてから選択してください。

## 確認項目

- [ ] Web リンクが青色で表示される
- [ ] 選択中リンクが反転ハイライトされる
- [ ] `#anchor` で見出しへジャンプする
- [ ] `O` でジャンプ前のスクロール位置に戻る
- [ ] 相対パスで別 md を開ける
- [ ] `./file.md#section` 形式が動作する
