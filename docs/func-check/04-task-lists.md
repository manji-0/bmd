# タスクリスト

[← 目次](./00-index.md)

GitHub 形式のタスクリストです。チェック状態のトグルはセッション内のみ有効で、ファイルには書き戻されません。

## 基本

- [ ] 未完了タスク A
- [x] 完了タスク B
- [ ] 未完了タスク C

## ネスト

- [ ] 親タスク
  - [x] 子タスク（完了）
  - [ ] 子タスク（未完了）

## 操作の確認

| 操作 | 期待結果 |
|------|----------|
| チェックボックスを左クリック | トグルされる |
| 対象行を画面上部に置き `x` | トグルされる |
| ファイルを再読み込み | 元の `[ ]` / `[x]` に戻る |

## マーカースタイル

環境変数 `BMD_CHECKLIST_STYLE` で変更できます。

| 値 | マーカー |
|----|----------|
| `unicode` | ☐ / ☑ |
| `emoji` | ⬜ / ✅ |
| `auto`（既定） | ターミナル自動判定 |

```bash
BMD_CHECKLIST_STYLE=unicode bmd docs/func-check/04-task-lists.md
```

## 確認項目

- [ ] 未完了・完了でマーカーが異なる
- [ ] クリックでトグルできる
- [ ] `x` で画面上部の項目をトグルできる
- [ ] ネストしたタスクも表示・操作できる
